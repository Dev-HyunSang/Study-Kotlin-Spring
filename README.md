## Getting Started
### Installation Kotlin on macOS(Homebrew)
```shell
brew install kotlin
```

### Installation OpenJDK on macOS(Homebrew)
```shell
brew update && brew upgrade
brew tap adoptopenjdk/openjdk
brew seacrh
brew search jdk
brew install --cask adoptopenjdk16
```
